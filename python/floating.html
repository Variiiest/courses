Function

result
math.acos(x) arc cosine of x
math.asin(x) arc sine of x
math.atan(x) arc tangent of x
math.sin(x) sine of x
math.cos(x) cosine of x
math.tan(x) tangent of x
math.asinh(x) arc hyperbolic sine x
math.acosh(x) arc hyperbolic cosine x
math.atanh(x) arc hyperbolic tangent x
math.ceil(x) return with x+1 or x
math.factorial(x) return x!
math.sqrt(x) x1/2
math.exp(x) return with ex
math.fmod(x,y) return with x//y
math.hypot(x,y) return with (x2+y2)1/2
math.e return with the value of x
math.log10(x) return with log10x
math.log1p(x) return with loge(1+x)
math.frexp(x) return with mantissa and integer
math.pi return with π
math.pow(x, y) xy
math.floor(x) return with x-1 or x
math.log(x, b) return with logbx
Don't use these function to learn basic programming.
