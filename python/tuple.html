Tuple

Tuple is an ordered sequence of objects.

Examples

Program to convert a tuple in list

>>> a='1','2','3','4'
>>> print(list(a))
['1', '2', '3', '4']

See also this program 

>>> alphabet="a","A","B","c","D"
>>> alphabet[2]
'B'
>>> alphabet[:2]    
('a', 'A')
>>> alphabet[2:]             
('B', 'c', 'D') 
>>> alphabet[2:],"vowel",alphabet[:2]
(('B', 'c', 'D'), 'vowel', ('a', 'A'))
>>> alphabet[2:]+("vowel",)+alphabet[:2]
('B', 'c', 'D', 'vowel', 'a', 'A')
