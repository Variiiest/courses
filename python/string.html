String

String is immutable data type.It can not be changed further. We can represent string in three type:

  • Single Quoted String
  • Double Quoted String
  • Triple Quoted String

String Program

Single Quoted String

>>> string='Anuman'
>>> print(string)
Anuman

Double Quoted String

>>> string="Anuman's"
>>> print(string)
Anuman's

Triple Quoted String

>>> text= """ 'Anumans' and "Anumans"  \ """
>>> text
' \'Anumans\' and "Anumans"  \\ '

String Joining Program

>>> #simple way
... a="do you want to learn"
>>> b=" python?"
>>> c=a+b
>>> print(c)
do you want to learn python?
>>> #nice way
... c=("do you want to learn" " python?")
>>> print(c)
do you want to learn python?

String Operation

Operator

result
string.capitalize() Capitalize a string
string.title() Convert in Title form
string.count(string1, start, end Counting of string1 in string
string.find(string1, start, end) finding of string1 in string
string.format(...) Return a copy of string
string1.index(string2, start, end) return the leftmost position of string2 in string1
string.isalnum() return with true if string is alphanumeric
string.isalpha() return with true if string is alphabatic
string.isdecimal() return with true if string is decimal
string.islower() return with true if all the characters of string are in lowercase
string.upper() Convert in uppercase
string.swapcase() Convert uppercase in lowercase and lowercase into uppercase
string.join(seq) concatenation of every item in the sequence
string.isspace() return with true if all characters are whitespace
string.isprintable() return with true if string is printable
string.replace(string1, string2, n) string1 replaced by string2 in the string for n times
string.split(string1, n) split string at string1 for n times
string.startswith(string1, start, end) return with true if string started with string1 from start index

String formating

Fill in the blanks:

Example-1
>>> s="Hello,'{0}' ,are you really excited to learn '{1}'?"
>>> s.format('Anurag','Python')
"Hello,'Anurag' ,are you really excited to learn 'Python'?"

Example-2
>>> s="Hello,'{}' ,are you really excited to learn '{}'?"
>>> s.format('Anurag','Python')
"Hello,'Anurag' ,are you really excited to learn 'Python'?"

Example-3
>>> s="Hello,{} ,are you really excited to learn {}?"
>>> s.format('Anurag','Python')
'Hello,Anurag ,are you really excited to learn Python?"

Concatenate strings: 

>>> a="Python"
>>> s='{} {} {} {}'
>>> s.format(a,"is","high-level","machine","language")
'Python is high-level machine'

Formating with List

>>> s = ["p", "a", "r", "a", "s"]
>>> string="a {0[1]} {0[2]}".format(s)
>>> print(string)
a a r
