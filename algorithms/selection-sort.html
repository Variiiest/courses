Selection Sorting

Selection sort is a sorting algorithm that has a quadratic running time complexity of O(n2), thereby making it inefficient to be used on large lists.Although selection sort performs worse than insertion sort algorithm, it is noted for its simplicity and also has performance advantages over more complicated algorithms in certain situations.

Algo

 SMALLEST (ARR, I, N, POS)
 1: [INITIALIZE] SET SMALL=ARR[I] 
 2: [INITIALIZE] SET POS=I 
 3: Repeat forJ=I+1 to N-1 
          IF SMALL>ARR[J] 
              SET SMALL=ARR[J] 
              SET POS=J 
          [END OF IF] 
        [END OF LOOP] 
 4: RETURN POS
 SELECTION SORT(ARR, N)
 1: Repeat s 2 and 3 for K=1 to N-1 
 2: CALL SMALLEST(ARR, K, N, POS) 
 3: SWAP A[K] with ARR[POS]
       [END OF LOOP] 
 4: EXIT

int smallest (int arr[], int k, int n)
{
  int pos = k, small = arr[k], i;
  for (i = k + 1; i < n; i++)
    {
      if (arr[i] < small)
	{
	  small = arr[i];
	  pos = i;
	}
    }
  return pos;
}

void selection_sort (int arr[], int n)
{
  int k, pos, temp;
  for (k = 0; k < n; k++)
    {
      pos = smallest (arr, k, n);
      temp = arr[k];
      arr[k] = arr[pos];
      arr[pos] = temp;
    }
}
