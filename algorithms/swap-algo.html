Swap Algo

Swap-Algo

Step 1: Input first number as A 
Step 2: Input second number as B 
Step 3: SET TEMP = A 
Step 4: SET A = B 
Step 5: SET B = TEMP 
Step 6: PRINT A, B 
Step 7: END

Swap-Algo Implemation in C


#include <stdio.h>
 int main()
{
  int x, y, temp;
  
  printf("Enter two integers\n");
  
  scanf("%d%d", &x, &y);
  
  printf("Before Swapping\n x = %d\ny = %d\n", x, y);
  temp = x;
  x = y;
  y = temp;
  
  printf("After Swapping\nx = %d\ny = %d\n", x, y);
  
  return 0;
}
