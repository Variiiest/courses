Newton method to find root

Newton's method, also known as the Newton–Raphson method, named after Isaac Newton and Joseph Raphson, is a root-finding algorithm which produces successively better approximations to the roots (or zeroes) of a real-valued function. The most basic version starts with a single-variable function f defined for a real variable x, the function's derivative f ′, and an initial guess x0 for a root of f. #include<stdio.h>
#include<math.h>
#define epsilon 0.00001
float f(float x)
{
    return x*x-2*x;
}
float df (float x)
{
    return 2*x-2;
}
int main()
{
    int iteration, max_iteration;
    float h, value, x1;
    printf("\nEnter value and maximum iterations\n");
    scanf("%f %d", &value, &max_iteration);
    for (iteration=1; iteration<=max_iteration; iteration++)
    {
        
        if (fabs(h) < epsilon)
        {
        	h=f(value)/df(value);
            x1=value-h;
            printf(" At max_iteration no. %d, x = %f\n", iteration, x1);
            printf("After % maximum iterations, root = %f\n", iteration, x1);
            return 0;
        }
        
    }
}
