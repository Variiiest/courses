Factorial Algo

Factorial Algo

Step 1: Input N 
Step 2: SET I = 1
        FACT = 1 
Step 3: Repeat Step 4 while I <= N 
Step 4: FACT=FACT*I
        I=I+1[ END LOOP]
Step 5: PRINT FACT
Step 6:END

Algo Implemation in C

        
#include <stdio.h>
int main()
{
  int n;
  int fact=1,i;
  printf("Enter a number to calculate factorial");
  scanf("%d",&n);
  printf("Factorial of %d=%d",n,fact); 
  for(i=1;i<=n;i++)
     {
      fact=fact*i;
     }
     printf("Factorial of %d=%d",n,fact); 
}
