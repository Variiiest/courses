File Operations

Operation Function Syntax
Opening fopen() fopen("file adress","r")
Closing fclose() fclose(file pointer)
Reading a character fgetc() fgetc(file pointer)
Reading a string fgets() fgets(file pointer)

File opening modes

Mode Use
"r" reading from a file
"w" writing to the file
"a" adding new contents at EOF(end of file)
"r+" reading, writing, modifying
"w+" reading, writing, modifying
"a+" reading , writing
