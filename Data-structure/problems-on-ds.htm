Problems

  • Write a method duplicate Stack that returns a new stack containing the same elements and in the same order as the stack specified in the parameter. The method should create a new stack and fill it with the same data elements as the given stack. (You do not need to duplicate the contents of the elements.). Before the method finishes, it must restore the contents of the original stack to its original state (same contents in the same order).
  • Write a program to convert an infix expression of integers into its equivalent postfix notation-using stack ADT.
  • Write a program to evaluate a postfix and prefix expression using a stack data structure.
  • Write a program to implement three stacks ADT using one array.
  • Use your stack to check whether a String is a Palindrome or not.
  • Sort the given set of elements present in three stack data structures and put them in a single stack, without using any other variable.
  • Write a program that takes a single positive integer (say N) as a command line argument, generates N random integers between 0 and 10, 000. Insert them (one by one) into a list in sorted order. Assume the list is initially empty.
      Example: Generated elements: 8, 5, 10, 1, . . . List: 8 → 5 8 → 5 8 10 → 1 5 8 10.
    1. Use the same process for storing the elements using : (a) an array; (b) a linked list;
    2. Run your program 6 times each for N = 100, 500, 1000, 2000, 3000, . . ., 10000.
      3. Print the sorted list to standard output, and the time taken (followed by a single tab, but no newline) to standard error.
    3. Find the average time taken for each value of N and for each implementation method given above.
    4. You are given a linked list of linked lists. Write a program to create a single linked list as as shown in the following example. Input: 
      2 → 6 →21 →25
↓ ↓ ↓ ↓
1 11 22 35
↓ ↓ ↓
8 50 47
↓ ↓
19 89
      Output 2 → 1 → 8 → 19 → 6 → 11 → 21 → 22 → 50 → 25 → 35 → 47→ 89
      5. Take input from a file with following format:
      • Line 1: # Number of lists
      • Line 2: Elements of List 1
      • Line 3: Elements of List 2
